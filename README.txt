Requires 

-CNelephante for reading sif files
-xcode in mac or Rtools in windows(might be a bit tricky)
-xtable
-Rsundials

After installing the package you will find a Network.sif in the 
odeBooleanR installation path inside a folder called examples. Copy the network to 
your current path.To test Run the sript example.R and you should see the simulated 
network values.The network parameters are inside the variable paramInfo. 
Feel free to tweak them as you want.
